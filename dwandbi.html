Data Warehousing & Business Intelligence Projects

Hot Topic-Oracle Upgrade:

- Client has performance issues with on-prem OLAP system as well as maintainace overhead of scheduling and executing jobs
- Solution: implementation of compute scalable snowflake cloud edw
- I build ETL for history conversion of omnichannel data from orders, sales, returns ,inventory from oracle system
- Loaded into to snowflake datawarehouse/datamart for the looker dashboard
- With the required performance boost achieved the client were able to decommision their legacy on-prem system

Staples-EDW:

Developed DWH with ETL frameworks in Snowflake.
- Client required a DWH for their retail data of sales, order, return, cost-to-serve, scm
- They had 5+ data sources, sources dump 1000+ unstructured data file in ADLS container at random frequency
- Developed data ingestion pipeline for history conversion as well as delta conversion, using snow-stream/snow-pipes/snow-tasks

David Yurman-Adobe Integration:

Integrated AWS for reporting system ETL.
- Client uses Salesforce to improve customer interactions and marketing campaigns. they relied in Hybris for managing customer data, marketing, and sales.
- As solution I used AWS, periodic lambda script excecution to trigger appflow to extract the salesforce data and load the snowflake archive db
- After each new archive my stored procedure tranforms the PII, orders, returns, sales data accordingly for the endpoint view
- Client gained scalable, high performance DWH for data-driven decision making and was able to decommision hybris
